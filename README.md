# Resumex

**Resumex** is an open-source resume builder. It gives you a live preview as you type, exports clean PDFs, and keeps your data entirely under your control.

## What it does

- **Live editor** — the preview updates as you fill out your details, no manual refresh needed.
- **Multiple templates** — Classic, Modern, and Minimal, each designed to pass ATS parsers cleanly.
- **PDF export** — generates a print-ready, single-page PDF directly from the browser.
- **Accounts & dashboard** — users can create, manage, and return to multiple resumes.
- **Admin panel** — view registered users, total resumes, and basic platform activity.
